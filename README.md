#shortly-angular

  > Your classic URL shortener! 
  
## Stack

Built using Angular, Node, Express, Passport, Mongoose, Gulp, Q and Bcrypt. 

####Future Deployment:
Docker, AWS EC2, AWS Lambda.

####Build tools: 
NPM scripting, Gulp, Babel, ESlint, Sinon and Phantom.

####Testing:
Mocha, Chai, Karma.
